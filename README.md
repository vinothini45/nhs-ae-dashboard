# NHS A&E Waiting Time Analyser 🏥

A live data dashboard showing 15 years of NHS England A&E waiting time data.

## 🔗 Live Dashboard
👉 [Click here to view](https://nhs-ae-dashboard-ekmvubelo4rltqyteavqdu.streamlit.app)

## 📊 What it shows
- % of patients waiting more than 4 hours
- Trends from 2010 to 2026
- Total visits per month
- Worst and average breach rates

## 🛠️ Built With
- Python
- Pandas
- Plotly
- Streamlit

## 📁 Data Source
[NHS England A&E Statistics](https://www.england.nhs.uk/statistics/statistical-work-areas/ae-waiting-times-and-activity/)

## 🚀 How to Run Locally
1. Clone this repo
2. Install requirements:
```
   pip install -r requirements.txt
```
3. Run the app:
```
   streamlit run read_data.py
```
